Digital Livestock Ecosystem for Kajiado County
AgriPulse is a mobile-first software ecosystem designed to empower pastoralist communities in Kajiado County, Kenya. By digitizing animal health records, providing real-time market intelligence, and facilitating logistics pooling, AgriPulse transforms traditional livestock management into a data-driven, sustainable economy.

🚀 Project Overview
Pastoralists often face challenges with fragmented health records, lack of price transparency, and high transport costs. AgriPulse addresses these via a Progressive Web App (PWA) that operates "offline-first" to accommodate the low-connectivity environments of rural ASAL (Arid and Semi-Arid Lands) regions.

Key Features
Digital Health Passport: A verifiable, QR-coded history of vaccinations and treatments for every animal.

Marketplace Intelligence: Real-time livestock prices from Kiserian and Ilbisil markets.

Logistics Pooling: A "shared economy" model for transporters to reduce the cost of moving animals to market.

Offline Synchronization: Data is stored locally on the device and syncs automatically when a network (3G/4G) is detected.

🛠️ Technical Stack
Backend: Django 4.2 (Python)

Frontend: React.js + Redux Toolkit (Progressive Web App)

Database: PostgreSQL + PostGIS (Geospatial data support)

APIs: Africa’s Talking (SMS Alerts), M-Pesa Daraja (Payments), OpenStreetMap (Logistics mapping)
